Medicina para Pobres

Que es "Medicina para Pobres"

Medicina para Pobres aun no es nada, pero me gustaria convertirlo en un entorno de estudio para estudiantes de las ramas de la medicina de todo el mundo. Un lugar donde puedan encontrar y compartir informacion, estudiar, competir, todo sin ningun costo.

Para ello necesito de la ayuda de varias personas, que busquen informacion, para colocarla en este sitio y que otros puedan verla, pero para esto necesito pagarle, por eso pienso colocar anuncios en esta web, para poder costearme los empleados.

Cosas que quiero implementar en este proyecto

  • Un lugar para leer libros, escuchar podcasts y ver videos de cada tema.
  • Un foro para preguntas y respuestas entre los usuarios.
  • Una plataforma para responder formularios, hacer examenes, participar en concursos, todo con el objetivo de insentivar al estudio mediante la competencia.
